部署、发布

修改_config.yml配置文件，支持多个配置信息，如果多个需要如下填写

  • 发布到github

    先进行安装npm install hexo-deployer-git --save

  • 发布到个人服务器（sftp）

    先进行安装npm install hexo-deployer-sftp --save

经过如上配置后，执行hexo -d后就会将生产的静态文件发布到git和服务器上。
